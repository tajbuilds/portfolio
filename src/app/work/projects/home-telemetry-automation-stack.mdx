## Overview

A self-hosted setup that collects telemetry, enriches it with automations, and surfaces the results in dashboards and alerts.

## Highlights

- MQTT-based event stream for device and vehicle data.
- Grafana dashboards with real-time alerting.
- Docker Compose deployment with easy backups and updates.

## Tech

Docker, MQTT, Grafana, Home Assistant.

## Outcome

A reliable personal observability stack with actionable alerts.
