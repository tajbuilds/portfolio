## Overview

A secure form handling pipeline that validates requests at the edge before forwarding to automation platforms. Built to stop spam and tampering while keeping latency low.

## Highlights

- Turnstile validation with replay protection.
- HMAC signature verification for tamper-proof payloads.
- Clean routing into n8n/Make webhooks with typed payloads.

## Tech

Cloudflare Workers, Turnstile, Webhooks, n8n, JavaScript.

## Outcome

Reliable form intake with reduced spam and safer downstream automation.
